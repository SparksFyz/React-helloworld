## Precompiling JSX

* Install babel
  * `npm install --global babel-cli`
  * Compile Directories
    * `babel src --out-dir build`
  * Piping Files
    * Pipe a file in via stdin and output it to script-compiled.js
    * `babel --out-file script-compiled.js < script.js`
